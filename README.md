## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/KyrlanAlanAlexandre/seomarket.git

# Navigate to the project directory
cd seomarket

# Install dependencies
npm install

# Start the development server
npm run dev
```
